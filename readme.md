# VisualElectricField *v0.5*
*Visualize and share electric fields online!*

VisualElectricField is (as the name suggests) a simulation that visualizes electromagnetic fields. It's a very simple simulation limited to indefinitely small point charges. You can create those simulations online and share codes, links, files and images for other people to reproduce and visualize your fields for themselves.

## Live demo
You can find an online version of VisualElectricField [here](https://sossinaydev.ch/VisualElectricField).
